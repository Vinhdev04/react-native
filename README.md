# Series Học React Native Từ Cơ Bản Đến Nâng Cao

Chào mừng các bạn đến với series học React Native từ cơ bản đến nâng cao! Series này sẽ cung cấp cho bạn kiến thức thực tế thông qua việc xây dựng các dự án cụ thể.

## 🚀 Lộ trình dự án

### 1. Dự án Cơ bản: Todo App
Ứng dụng quản lý công việc hàng ngày, giúp bạn làm quen với các khái niệm cốt lõi của React Native:
- Các Component cơ bản (View, Text, TextInput, TouchableOpacity,...)
- Quản lý State với Hooks
- Render danh sách với FlatList
- Style ứng dụng bằng StyleSheet
- Tương tác người dùng (thêm, xóa, sửa công việc)

### 2. Dự án Nâng cao: Food App
Ứng dụng đặt đồ ăn hoàn chỉnh, giúp bạn nâng cao kỹ năng xây dựng app thực tế:
- Điều hướng mượt mà với React Navigation
- Quản lý State phức tạp (Redux Toolkit, Zustand, hoặc Context API)
- Gọi API và xử lý dữ liệu (Axios / Fetch)
- UI/UX nâng cao và Animation
- Tối ưu hóa hiệu năng ứng dụng
